# eccentric-GWTC-1
A repository for holding the config and prior files and results of my eccentric analysis of [GWTC-1](https://www.researchgate.net/publication/329362628_GWTC-1_A_Gravitational-Wave_Transient_Catalog_of_Compact_Binary_Mergers_Observed_by_LIGO_and_Virgo_during_the_First_and_Second_Observing_Runs) gravitational wave events from merging binary black holes.
